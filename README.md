# Blender-Space-Animation

Title: Space Orbit Animation | Blender Project

Description:
Created a short space-themed animation using Blender. The project features an Earth background, a space environment, and a camera orbiting around the scene to create a cinematic effect. Space station assets were obtained from free online resources, while the scene setup, camera movement, lightning, adujesting nodes, Zoom effects, composition, and final rendering were completed by me.

Note: The original Blender project files were lost while i restart my Pc, but the final rendered video has been preserved.
